1. Introdução

Este plano de teste tem por objetivo testar todas as classes do Projeto Appanela

2. Escopo de Teste 

Todas as classes do projeto foram testadas. São elas: Categoria, Histórica, Ingrediente, Parceiro, ProdutoParceiro, Receita, Usuario, UsarioPremium, Visitante.


3. Fora do Escopo de Teste 

Interfaces não testadas.

4. Considerações de Infraestrutura 

As plataformas utilizadas para os testes foram: Computador e a IDE NetBeans.

5. Premissas 

Todas os atributos foram considerados como private e os métodos foram considerados como públicos.

6. Riscos 

Como não há banco de dados, as regras de negócios vinculadas a este requisito não foram testadas.

7. Conclusão 


Todos os testes foram realizados. De acordo com a IDE NEtbeans, os testes foram concluídos com sucesso, sem apresentar nenhum erro. Assim, nota-se que o código-fonte está sendo construído com êxito.
